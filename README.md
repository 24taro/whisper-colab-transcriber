# 🎙️ Whisper Colab Transcriber

**OpenAI Whisper large-v3を使用した高精度音声書き起こしツール**

プログラミング知識不要で、誰でも簡単に音声ファイルをテキストに変換できるGoogle Colabツールです。

## ✨ 特徴

- 🎯 **高精度**: OpenAI Whisper large-v3モデル使用
- 🚀 **簡単操作**: ワンクリックで完結
- 📱 **多形式対応**: MP3、M4A、AAC対応
- 🧹 **高品質**: 不要フレーズ除去、句読点改善
- 💾 **即時ダウンロード**: 処理完了後すぐにダウンロード

## 📋 使い方

### 1. ノートブックをダウンロード
```bash
# GitHubからクローン
git clone https://github.com/24taro/whisper-colab-transcriber.git
```
または、[index.ipynb](index.ipynb) を直接ダウンロード

### 2. Google Colabにアップロード
1. [Google Colab](https://colab.research.google.com/) にアクセス
2. 「ファイル」→「ノートブックをアップロード」
3. ダウンロードした `index.ipynb` を選択

### 3. GPU設定（重要！）
1. 「ランタイム」→「ランタイムのタイプを変更」
2. 「ハードウェア アクセラレータ」を **「T4 GPU」** に変更
3. 「保存」をクリック

### 4. 実行
1. 「ランタイム」→「すべてのセルを実行」をクリック
2. セットアップ完了後、音声ファイルをアップロード
3. 自動で書き起こし開始 → 完了後ダウンロード

## 🔄 複数ファイル処理時の注意

**2回目以降の実行前に必ず実行：**
1. 「ランタイム」→「ランタイムを再起動」
2. 再起動後、再度「すべてのセルを実行」

*これを行わないとGPUメモリ不足エラーが発生します*

## 📊 処理時間の目安

| 音声長 | 処理時間 |
|-------|---------|
| 5分   | 約2分   |
| 30分  | 約10分  |
| 60分  | 約20分  |

## 💡 Tips

- **ブラウザ**: Google Chrome推奨
- **ファイルサイズ**: 100MB以下推奨
- **音質**: クリアな音声ほど精度向上
- **言語**: 日本語・英語自動判定

## 🛠️ 技術仕様

- **AI モデル**: OpenAI Whisper large-v3
- **必要環境**: Google Colab (T4 GPU)
- **対応形式**: MP3、M4A、AAC
- **出力形式**: UTF-8テキストファイル
